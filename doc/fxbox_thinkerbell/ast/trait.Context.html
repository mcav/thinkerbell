pub trait Context: Serialize + Deserialize + Default { }

A manner of representing internal nodes.

Two data structures implement Context:

  • UncheckedCtx, designed to mark the fact that a script has not been compiled/checked yet and must not be executed;
  • compile::CompiledCtx, designed to mark the fact that a script has been compiled and can be executed.

Implementors

  • impl Context for UncheckedCtx
  • impl<Env> Context for CompiledCtx<Env> where Env: Serialize + Deserialize
